 
 
Tables and the Border Attribute 
To display a table with borders, you will use the border attribute. 
 
This Code  Would Display  
<table border="1"> 
<tr>  
<td>Row 1, cell 1</td>  
<td>Row 1, cell 2</td>  
</tr>  
</table>
