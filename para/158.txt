Headings in a Table 
Headings in a table are defined with the <th> tag. 
 
This code  Would Display  
<table border="1">  
<tr>  
<th>Heading</th>  
<th>Another Heading</th>  
</tr>  
<tr>  
<td>row 1, cell 1</td>  
<td>row 1, cell 2</td>  
</tr>  
<tr>  
<td>row 2, cell 1</td>  
<td>row 2, cell 2</td>  
</tr>  
</table>
